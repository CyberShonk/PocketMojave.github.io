# Shonk's Pocket Mojave

Static GitHub Pages site for Shonk's Pocket Mojave, a Fallout: New Vegas on Android guide for GameNative and Droid Mod Loader.

## Pages

- `index.html`
- `setup.html`
- `mods.html`
- `load-order.html`
- `avoid.html`
- `changelog.html`
- `report.html`

The site uses the same layout structure and CSS style as Shonk's Pocket Wastelands while keeping the Pocket Mojave guide content.
