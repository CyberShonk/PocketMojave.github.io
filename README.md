# PocketMojave.github.io

Static GitHub Pages files for **Shonk's Pocket Mojave**.

## Files

- `index.html`
- `setup.html`
- `mods.html`
- `load-order.html`
- `avoid.html`
- `report.html`
- `styles.css`
- `copy.js`
- `.github/ISSUE_TEMPLATE/pocket-mojave-report.yml`

GitHub Pages can serve this directly as a static site. No Jekyll theme or plugin is required.

## Issues

This site includes a GitHub issue form for Pocket Mojave reports. Make sure the repository has Issues enabled:

1. Open the repo on GitHub.
2. Go to Settings.
3. Under Features, enable Issues.
4. Push the `.github/ISSUE_TEMPLATE` folder.
5. Open the Issues tab and click New issue.
