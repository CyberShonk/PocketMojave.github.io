Mod List

Install the sections in order. Use the listed main file unless the notes say otherwise. Optional files are only listed when they are worth knowing about for this Android guide.

Recommended mod authors

Qolore7

Recommended for modern New Vegas fixes, UI improvements, quality-of-life work, and clean alternatives that usually fit this style of list well.

Stentorious

Recommended for modern New Vegas visuals, lighting, animation, and ESPless-style improvements that are worth testing on Android.

NVTF texture pool note
For this guide, enable the texture pool behavior in NVTF.ini. Start with bModifyDirectXBehavior=1 and bUseDefaultPoolForTextures=1. If you see fullscreen, alt-tab, or container-specific weirdness, revert only these settings and retest.

Installer rule

If DML does not show the right installer choices yet, install the mod on PC with MO2, pick the options you need, zip the clean installed output, then install that zip in DML.

Check meshes, animations, and audio

  1. Deploy in DML.
  2. Look for missing meshes, purple textures, broken hands, broken Pip-Boy placement, and missing audio.
  3. Fire a few weapons indoors and outdoors.
  4. If using third person fixes, test both first person and third person.
