Reports

Reports should go to the Pocket Mojave website repo so they are easy to track and update.

Submit a report

Open a new issue here: Submit a Pocket Mojave issue

Use the issue form if it appears. Attach the DML support report, plugin list, load order, and crash logger output if you have them.

Performance reports

  1. Use GameNative's overlay first for a quick FPS check.
  2. Use the same route and weather each time.
  3. Only switch to DXVK HUD if you need frame time, draw call, pipeline, or memory details.
  4. Turn the DXVK HUD off after the benchmark.
DXVK_HUD=fps,frametimes,drawcalls,pipelines,memory

Good report rules

  • Say the last mod or setting changed before the problem appeared.
  • Attach files instead of screenshots when possible.
  • Do not report a modded crash if vanilla GameNative never passed the vanilla check.
  • Do not change five things and then report only the crash.
